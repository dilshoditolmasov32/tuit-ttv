# MindAR Image Target

Place the compiled faculty marker here:

```txt
public/assets/targets/faculty-logo.mind
```

Generate it from the faculty logo or poster with the MindAR image compiler:

```bash
npx mind-ar-js image-target compile --input ./assests/logo.jpg --output ./public/assets/targets/faculty-logo.mind
```

Use a high-contrast printed logo or poster, avoid glossy reflections, and keep the same image for the compiled target and the physical marker shown to the camera.
